=== Sync Product From Amazon ===


== Description ==
If you want to display products from your Amazon store on WordPress posts or pages, this plugin is perfect for you. It fetches product details from Amazon and allows you to save them into posts or pages with ease.

The plugin supports both shortcodes and Gutenberg blocks to display products on the frontend. All you need is the product's ASIN number, which you can find on the Amazon product page.

Additionally, this plugin allows you to create individual posts or pages to showcase Amazon products. You can either add products manually or import multiple products in bulk using their ASIN numbers.

== Key Features ==
* Fetch and display product details directly from Amazon using ASIN numbers.
* Supports both shortcodes and Gutenberg blocks for easy integration into your posts or pages.
* Create individual posts or pages to showcase products.
* Option to manually add products or import multiple products in bulk using ASIN numbers.
* User-friendly interface for managing and displaying products.

== How It Works ==
1. **Install the Plugin:** Upload the `sync-product-from-amazon` directory to the `/wp-content/plugins/` directory and activate it through the 'Plugins' menu in WordPress.
2. **Add Products:** You can either manually add products by entering their ASIN numbers.
3. **Display Products:** Use the provided shortcode or Gutenberg block to display the selected products on your desired posts or pages.


== Installation ==
1. Upload the `sync-product-from-amazon` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the provided shortcode or Gutenberg block to display your Amazon products.

== Frequently Asked Questions ==
= How do I find the ASIN number? =
You can find the ASIN number on the Amazon product page, usually located in the product details section.

= Can I display multiple products? =
Yes, you can display multiple products by either adding them manually or importing them in bulk using their ASIN numbers.
